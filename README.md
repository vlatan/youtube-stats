# YouTube Metadata

Supply a YouTube video ID to get some of the essential info about the video.

Build CLI and WEB binaries and start a webserver.
```
make run
```
Access the web app on `localhost:8080`.

Use the CLI.
```
./cli jNQXAC9IVRw
```

(Re)start the webserver.
```
./ui
```

Clean up.
```
make clean
```
